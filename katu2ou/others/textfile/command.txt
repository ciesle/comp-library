<エディタ>
    c++のcoderunnerを走らせる：control + option + N
    c++のcoderunnerを強制中断する：control + shift + Q
    コンパイル+入出力テスト:control + option + N
    コードテストをする:control + T
    コードを提出する:control + S
    コード実行の中断:control + C
    次のブックマークの行に飛ぶ: option + command + L
    前のブックマークの行に飛ぶ: option + command + J
    その行をブックマークに追加: option + command + K
    コードの整形: Shift + Option + F
    c++ 高速なコンパイル: g++ -O3 filename.cpp -o filename //Oは大文字オー
        → ./<filename>
    コマンドパレット: shift + command + P

<ショートカット集>
    範囲選択: shift + 矢印
    コード内の検索: command + F
    コード内の文字の置換: option + command + F
        - 選んでいる範囲のみに置換対象を絞る: option + command + L
        - 選択範囲を全置換: command + enter
    コードの最上(下左右)に移動: command + 矢印
        - そこまでの範囲を全選択: command + shift + 矢印

<ターミナル>
    コンパイル: g++ <コンパイルオプション> main.cpp(コンパイルするファイルの名前) -o a.out(作成されるファイルの名前)
    実行時間測定: コンパイル->timeをつけて実行
    入力ファイルと出力ファイルを指定: ./main.out < input.txt > output.txt

<UNIXコマンド集>
    pwd : 現在ディレクトリのフルパス表示
    cd (dir) : ホームディレクトリ(指定したディレクトリ)へ移動 
    mkdir (dir) : 新規ディレクトリを作成
    rmkdir (dir) : 空のディレクトリを削除
    rm -r (dir) : 指定ディレクトリを中のファイルごと削除
    ls (dir) : ディレクトリのファイル一覧表示
        -a : 隠しファイルも含めて
        -l : 詳細情報も含めて
        -R : ディレクトリ下層まで辿って表示
    cd - : 直前のディレクトリへ移動
    cat (file) : ファイルの内容を表示
    cat (fileA) (fileB) > (file) : ファイルを結合
    rm (file) : ファイルを削除
        -i : ファイル削除の確認する
    cp (fileA) (fileB) : ファイルAをファイルBにコピー
    touch (file) : 空のファイルを作成
    touch (既存file) : ファイル更新日時を変更
    file (file) : ファイルの種類を調べる
    find (dir) : 指定ディレクトリ以下のファイルを列挙
        -name (文字列) : 指定した文字列に一致するファイル・ディレクトリを検索
    history : コマンド実行履歴を表示
    kill : そのプロセスを強制終了
        -9 : 何がなんでも強制終了
    shutdown : シャットダウンする
        -r : シャットダウンして再起動する
    reboot : 再起動
    who : 現在ログインしているユーザを表示
    whoami : 現在のログイン名を表示
    passwd (user) : 指定ユーザのパスワードを設定
    echo (文字列) : その文字列を標準出力に表示
    date : 現在日時・時刻の表示
    ? : 任意の一文字
    * : 0文字以上の文字列
