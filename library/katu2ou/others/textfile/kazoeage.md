# いろいろな数え上げ

## 行列木定理と Cayley の定理

### 行列木定理

無向グラフ `G` に対して、`G` の全域木の個数は、`G` のラプラシアン行列の任意の余因子と等しい。
ここでいう余因子は、`(-1)^{i+j}` 倍を含んだものを指す。

ここで、無向グラフ `G` のラプラシアン行列とは、各行・各列がそれぞれ `G` の頂点に対応していて、次の値を入れた対称行列のこと。

- 対角成分: その頂点の次数
- 非対角成分: 辺がある部分に `-1`、そうでない部分に `0`

### Cayley の定理

`n` 個のラベル付き頂点を持つ木の個数は、次で表される。

$$
n^{n-2}
$$

## Cauchy-Frobenius の補題

`X = {1, 2, ..., n}` 上の置換群で、`k` 個の軌道を持つものを `G` とする。
このとき、`G` の置換による不動点の平均値は `k` である。

言い換えると、`G` による対称性で同一視した場合に相異なる場合の数は `k` である。

$$
\frac{1}{|G|} \sum_{u \in G} |\{x \in X \mid u(x) = x\}| = k
$$

## 木の数え上げ

### Prufer コード

頂点集合 `V = {1, 2, ..., n}` からなり、頂点 `n` を根とする木 `T` を考える。
`T` に対する Prufer code は、頂点列 `(y_1, y_2, ..., y_{n-2})` として次のように定義される。

以下の操作を `k = 1, 2, ..., n - 1` について順に行う。

1. `T` のラベル最小の葉を `x_k`、その親を `y_k` とする。
2. `T` を `T - x_k` に置き換える。

このとき、`n` 頂点ラベル付き木全体から `V^{n-2}` への写像 `P` は全単射である。
したがって、木の個数を数えるには対応する Prufer code の個数を数えればよい。

例として、`n` 頂点ラベル付き木であって、各頂点 `i` の次数が `d_i` であるような木の個数を考える。
次数 `d_i` の頂点 `i` は Prufer code にちょうど `d_i - 1` 回出てくるので、各 `i` が `d_i - 1` 個現れる数列を数えればよい。

## Pick の定理

各頂点が格子点にあり、かつ内部に穴の空いていない多角形を考える。

- `i`: 多角形の内部にある格子点の個数
- `b`: 辺上にある格子点の個数

このとき、この多角形の面積 `S` は次で表される。

$$
S = i + \frac{b}{2} - 1
$$

`i` と `S` は比較的簡単に分かることが多く、そこから `b` を求めることができる。

## 分割数

自然数 `n` を `k` 個の `0` 以上の整数の和として表す方法の個数を `P(n, k)` とする。

次の漸化式が成り立つ。

$$
P(n, k) = P(n, k - 1) + P(n - k, k)
$$

右辺の意味は次の通り。

- 第 1 項: `n` を `k - 1` 個の和として表す。これは、少なくとも 1 個は `0` である `k` 個の和とみなせる。
- 第 2 項: `n` を `k` 個の `1` 以上の整数の和として表す。

## Stirling 数

`n` 個の区別できるものを、`k` 個の区別できない箱にそれぞれ 1 個以上入れる方法の個数を `S(n, k)` とする。
これは grouping の個数である。

漸化式:

$$
S(n, k) = S(n - 1, k - 1) + kS(n - 1, k)
$$

包除原理による式:

$$
S(n, k) = \frac{1}{k!} \sum_{i=1}^{k} (-1)^{k-i} \binom{k}{i} i^n
$$

下降階乗との関係:

$$
f_k(x) = x(x - 1)\cdots(x - k + 1)
$$

$$
x^n = \sum_{k=1}^{n} s(n, k) f_k(x)
$$
