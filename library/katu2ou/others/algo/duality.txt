

-- <Monge性> -- 
    - 行列AがMonge性を満たすとは，
         A_{i1,j1} + A_{i2,j2} <= A_{i1,j2} + A_{i2,j1} (近づいた方が大きいイメージ)

    - 辺重み c:E->Z がMongeであるとは，
        0<=i<j<k<l<n  ->  c(i,l) + c(j,k) >= c(i,k) + c(j,l)  (劣モジュラと逆向きの関係?)
    
    - A_{1,j_2} - A_{1,j_1} >= A_{2,j_2} - A_{2,j_1} >= ... (iが小さいほど右-左が大きい)

    -- [性質] --
        - 行列AがMongeであることは，A_{i,j}+A_{i+1,j+1} <= A_{i+1,j} + A_{i,j+1} であることと同値

        - 転置してもMongeで，Aの部分行列もMonge

        - MongeならTotally monotone (monotone:各行ベクトルのargmin(行の中で最小の要素の列番号))が広義単調増加
            (ここで，行の中に同じ値が存在する時は，Aの各要素に列番号を付加し，その辞書順で大小を定める)
            - 逆はわからない?(大体は逆も成立するらしい https://hackmd.io/@tatyam-prime/monge1)

        - Aがtotally monotone であるとは，行i,i+1,j_1,j_2からなる部分行列が常にmonotoneであることと同値
        - これは，任意の二列に対して，要素の大小関係を上の行から見て行った時，上から何行かが<で，それ以降
        全て>となっていることと同値．

        - 一つの行・列に値を加算してもMonge

        -　A,Bを合成(c_{ij} = min_k (a_{ik} + b_{kj}))してもMonge

        - Monge 性は和と非負スカラー倍で閉じている

        - 上三角・下三角だけMonge性を満たすならその範囲でMonge性が適用できる

        ex) 下に凸な関数f(Z->R)に対して，A_{i,j} = f(j-i) はMonge

        ex) 区間の長さの二乗(j-i)^2, 区間和(a_{i,j}=x_i+...+x_{j-1})(とその二乗), 区間minはMonge

        ex) a_{ij} = -ij はMonge

        ex) Xが単調増加，Yが単調減少でa_{i,j} =x_i*y_j はMonge

        ex) Monge Aと広義単調増加関数f(N->N)に対して，b_{i,j}=a_{i,f(j)}とするとBはMonge

        ex) a,bが広義単調増加な整数列ならA_{i,j}=(a_i-b_j)^2としたときAはMonge

        ex) a_{i,j} = b_i and b_{i+1} and ... and b_{j-1} としたときAはMonge

    -- [d辺s-t最短路を高速に求める] -- (https://noshi91.github.io/algorithm-encyclopedia/d-edge-shortest-path-monge#fn:Aliens)
        - 辺の重みがMonge性を満たすような完全DAG(i<jなる(i,j)全てを辺に持つ)について，辺をちょうどd本使う条件下での
        (0 N-1)最短路長をO(N log(max |c(e)|)) で求める (ラグランジュ双対問題との強双対性による)
        (c(e)は負の辺を持っていても良い(dが固定なので全部にoffsetを足して非負にできるし))

        - この問題は制約付き最適化であり，、そのラグランジュ双対問題との強双対性がcのMonge性から成り立つ．
        この強双対性を使うことで，ラグランジュ緩和問題をO(log(max |c(e)|))回解く問題に帰着される．

        - ラグランジュ緩和問題は，全ての辺の重みをaだけ大きくした場合の(辺の数の制約のない)最短路問題となる．
        cがMongeなら，一様にaを加算したものもまたMongeである．Monge性を満たす辺重みの最短路はLARSCH Algorithm
        を使ってO(N)で計算できる．

        - 一般に，ラグランジュ緩和問題を用いる発想をAlien DPと呼ぶことがある．

        (ラグランジュ緩和問題)
            - 制約の一部を除去し，除去した制約について違反した量を一次のペナルティとして目的関数に組み込んだ問題を
            ラグランジュ緩和問題と呼ぶ．(ラグランジュ緩和問題の解は，元の問題の解の下界を与えている)
            
            ex) 辺の個数dでの最短路問題を考える時，
                min c(P) (コスト和の最小) ≧ min (c(P) + a((Pの辺の個数)-d)) 
            
            ここで，aを整数とした時，左辺のmax_{a} の値(これを求めるのがラグランジュ双対問題) もmin c(P)以下であるが，
            <<この「以下」は「イコール」になることが示せる(強双対性) 一般にLPのときは等号成立>>

        (双対問題の解き方)
            - L: a -> min(c(P)+a((Pの辺の個数)-d))は一次関数群のminであるから，上に凸である．よって，
            Lの最大値は三分探索を用いて計算でき，その求値がmin c(P)である．

            - aが与えられた時L(a)を計算する．辺重みc_a:E->Zをc_a(e)=c(e)+aによって定めると，
                L(a) = min(c(P)+a((Pの辺の個数)-d)) = min(c(P)+a((Pの辺の個数)-ad))
                    = -ad + min c_a(P)
            であるから，c_aについての最短路長を計算することでL(a)の値を得られる．

            - cがMongeであるから，c_aもまたMongeである．辺の重みがMonge性を満たすDAGの最短路の
            計算はLARSCH Algoを用いてO(N)で行うことができる．さらに，条件が良い時はConvex Hull Trick
            を用いることで同じくO(N)で最短路を計算できる．

            - 三分探索の下界として-3max|c(e)|を，上界として3max|c(e)|を用いることができる．

        - 制約がないときに簡単だが，制約のせいで解きにくい場合に使えるか試してみる

    -- [monotone minima] --
        - 行列Aがmonotoneなら，Aの行最小値はO(n+mlog m)で求まる
        (行列の中央の行の最小値を求めると，monotone性よりその左下と右上の部分に他の行の最小値は存在しない
        から，半分のサイズの問題*2に分割できる)

        - SMAWK algorithm でO(n+m)で解ける

        - オンラインMonge行最小値問題は LARSCH algorithm を用いると実はO(n+m)で解ける

    -- [はやい区間DP] --
        - AがMongeかつ単調(区間が長いほど大きい)とき，区間DPがO(n^2)でできる
